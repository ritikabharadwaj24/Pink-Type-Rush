# Pink Pixel Type Rush Arcade

This is a simple typing speed game built using HTML, CSS, and JavaScript.  
I created this project to practice my frontend development skills and improve my understanding of JavaScript logic and DOM manipulation.

# About the Project

Pink Pixel Type Rush Arcade is a browser-based typing game where users type displayed sentences within a time limit. The game tracks speed (WPM), accuracy, and score.

The main goal of this project is to make typing practice more interactive and fun using a game-like interface.

# Features

- 60-second typing challenge
- Multiple difficulty levels (Easy, Medium, Hard)
- Real-time score tracking
- WPM (Words Per Minute) calculation
- Accuracy percentage system
- Progress bar with visual feedback

# Technologies Used

- HTML
- CSS
- JavaScript

# How to Run This Project

1. Download or clone this repository
2. Open the project folder
3. Run `index.html` in any browser

No additional setup is required.
